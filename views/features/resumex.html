Section Types

Personal Info Header

Name, title, photo (circle/square/none), contact details, social links with icon auto-detection (LinkedIn, GitHub, Twitter).

Work Experience

Multi-entry timeline with company, role, dates, location, and bullet-point achievements. Date range picker with "Present" option.

Education

Institution, degree, field, GPA (optional), honors, relevant coursework list, and thesis title fields.

Skills & Proficiency

Tag-based skill list with optional proficiency bars (beginner/intermediate/expert) or star ratings. Category grouping.

Projects & Portfolio

Project cards with title, description, tech stack badges, live URL, and GitHub link. Thumbnail support.

Certifications & Awards

Certification name, issuer, date, credential ID, and verification URL. Badge display mode available.

Languages

Language name with proficiency level (A1–C2 / Native). Visual flag icons support.

Volunteer / Interests / Custom

Free-form custom sections with any label. Perfect for volunteer work, publications, patents, or hobbies.

Import resume data from LinkedIn profile URL via scrape-safe parser. Maps profile sections to ResumeX fields automatically.

Full support for jsonresume.org schema. Import and export in standardized format for interoperability.
