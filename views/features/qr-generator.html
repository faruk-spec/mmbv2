Core Features

Everything in One Place

All fundamental QR code capabilities — battle-tested and production-ready.

QR Content Types

URL / Website Link

Encode any URL — static or dynamic with redirect tracking. Supports HTTP, HTTPS, and custom deep-link schemes.

Most Used

vCard / Contact

Encode full vCard 3.0 & 4.0 data: name, phone, email, address, photo URL, company, and social handles.

Wi-Fi Credentials

SSID, password, encryption type (WPA/WEP/None). Hidden network support. One-tap connect for guests.

Email / SMS

Pre-fill recipient, subject, and body for email; pre-fill number and message for SMS. Works across all OS.

GPS Location

Latitude/longitude with optional altitude, Google Maps URL fallback, and Apple Maps support.

Phone / FaceTime / WhatsApp

Direct dial, WhatsApp chat, FaceTime audio/video — all encapsulated in a single scannable code.

Calendar Event (vCal)

Create iCal events with title, date/time range, location, description, organizer, and attendees list.

PDF / File Link

Point to any file (PDF, DOCX, MP4) with optional inline viewer, download prompt, and expiry control.

Crypto Payment

Bitcoin, Ethereum, and major altcoins — encode wallet address, amount, label, and memo fields.

Customization Engine

Full Color Control

Foreground, background, gradient fills (linear/radial), and per-module color. Eye dropper + hex/RGB/HSL input.

Module Shapes

Square, rounded, dots, classy, classy-rounded, extra-rounded — applies to data cells and corner finders independently.

Center Logo Embed

Upload SVG, PNG, or JPEG logo. Adjustable size, padding, background shape, and auto error-correction boost to H (30%).

Frame & CTA Labels

Add decorative frames with call-to-action text ("Scan Me", custom label), icon overlays, and border styles.

Error Correction Levels

L (7%), M (15%), Q (25%), H (30%) — automatically upgraded when logo embed is detected.

Size & Quiet Zone

Output pixel size from 64px to 4096px; quiet zone modules configurable from 0 to 10.

Dynamic QR Codes

Real-Time Destination Edit

Change the QR destination URL without reprinting. Redirect rules updated instantly, zero downtime for printed materials.

Dynamic

A/B Redirect Split Testing

Route scan traffic between multiple URLs at configurable percentages. Built-in conversion tracking per variant.

Scheduled Redirects

Define active windows with start/end timestamps. Auto-expire to fallback URL after campaign ends.

Scan Limit & One-Time QR

Hard cap on total scans or unique-IP scans. Perfect for gated content, tickets, and single-use vouchers.

Bulk & Automation

Batch Export (ZIP)

Folder / Campaign Organizer

Group QR codes by campaign, event, location, or department. Nested folder support with color labels.

Template Library

Save style presets (colors, logo, frame) as reusable templates. Share templates across team members.

Webhook Callbacks

Print-Ready PDF Export

Generate CMYK-optimized print PDFs with bleed/trim marks, label sheets (Avery layouts), and custom DPI.
